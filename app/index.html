WHAT I DO

I'm Memona Madi, a self-taught creative user interface developer currently based in London, UK.

As part of the Cedar DCX team I do front-end development work for Tesco, Danone and British Airways, amongst others.

I have a strong experience working with HTML(5), CSS(3), Sass, Javascript and jQuery and like to use a modern workflow in my work, using tools like Grunt.js, Bower and Git.

I try to learn something new everyday and build up my skills by creating side projects and going to tech events in London.
